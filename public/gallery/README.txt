DROP KURT'S REAL PHOTOS HERE
============================
1. Save photos from https://www.instagram.com/malpassbrickwork/ into this folder
   (e.g. wall-01.jpg, extension-01.jpg).
2. Open  src/lib/gallery.ts  and change each `src` to "/gallery/your-file.jpg".
3. Set the right `category` and a short `alt` description for each.
That's it — the gallery, filters and lightbox update automatically.
